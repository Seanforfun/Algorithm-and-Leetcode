## 26. Remove Duplicates from Sorted Array
### Thinking:
* Method1: Save a temp value to compare.
```Java
	class Solution {
	    public int removeDuplicates(int[] nums) {
	        int result = 1;
	        int len = nums.length;
	        if(0 == len)    return 0;
	        int temp = nums[0];
	        for(int i = 1; i < len; i++){
	            int cur = nums[i];
	            if(temp != cur){
	                nums[result++] = cur;
	                temp = cur;
	            }
	        }
	        return result;
	    }
	}
```
